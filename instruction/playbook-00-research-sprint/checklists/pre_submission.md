# Pre-Submission Checklist

> Final verification before clicking submit

---

## 1. Paper Content

### Claims Match Results
- [ ] All numbers in abstract match tables
- [ ] All claims are supported by experiments
- [ ] Improvement percentages are calculated correctly
- [ ] No stale numbers from earlier runs

### Writing Quality
- [ ] Abstract is self-contained
- [ ] Contributions clearly stated in intro
- [ ] Method is reproducible from description
- [ ] Limitations acknowledged honestly

### References
- [ ] All citations are in bibliography
- [ ] No orphaned references (in bib but not cited)
- [ ] Citation years and venues correct
- [ ] Author names correct

### Figures and Tables
- [ ] All figures referenced in text
- [ ] All tables referenced in text
- [ ] Figure quality is high (not blurry)
- [ ] Tables fit within margins
- [ ] Captions are informative

---

## 2. Formatting

### Page Limits
- [ ] Within page limit for main content
- [ ] References don't count (if allowed)
- [ ] Appendix follows venue rules

### Template Compliance
- [ ] Using correct template/style file
- [ ] Font sizes correct
- [ ] Margins not modified
- [ ] No forbidden packages

### Anonymization (if double-blind)
- [ ] No author names in PDF
- [ ] No identifying information
- [ ] Acknowledgments removed/anonymized
- [ ] No "our previous work" self-citations
- [ ] GitHub links anonymized or removed

---

## 3. File Preparation

### Main Document
- [ ] Single PDF file
- [ ] All figures embedded
- [ ] PDF opens correctly
- [ ] No compilation warnings in LaTeX

### Supplementary (if allowed)
- [ ] Formatted correctly
- [ ] Clearly labeled sections
- [ ] Self-contained

### Code/Data (if required)
- [ ] Repository is clean
- [ ] README is clear
- [ ] License included
- [ ] Anonymized if needed

---

## 4. Submission Portal

### Account Setup
- [ ] Account created on submission system
- [ ] Email verified
- [ ] Profile complete

### Submission Form
- [ ] Title entered correctly
- [ ] Abstract entered correctly
- [ ] Authors and affiliations correct
- [ ] Keywords/topics selected
- [ ] Conflicts of interest declared
- [ ] PDF uploaded and rendered correctly

### Deadlines
- [ ] Know exact deadline (including timezone)
- [ ] Submitting with buffer time
- [ ] Know if there's a revision period

---

## 5. Final Read-Through

- [ ] Read paper start to finish
- [ ] No TODO comments left in text
- [ ] No placeholder text
- [ ] Grammar and spelling checked
- [ ] Consistent terminology throughout

---

## Last-Minute Fixes to Avoid

- [ ] DON'T make major changes in final hours
- [ ] DON'T update numbers without re-verifying
- [ ] DON'T add new sections
- [ ] DON'T change the abstract significantly
- [ ] DO fix typos and minor formatting

---

## Submission Confirmation

After submitting:
- [ ] Received confirmation email
- [ ] PDF in portal matches your file
- [ ] All fields look correct
- [ ] Noted submission ID
